React Cheatsheet

React.js

  • React – A JavaScript library for building user interfaces.

  • The most popular framework today.

  • Free and open source

Fundamentals

  • React makes it painless to create interactive UIs

  • Component based

  • JSX is optional and not required to use React.

Hooks

  • Hooks are functions that allow you to maintain a state and react to components' lifecycles.

  • Were created to simplify the code.

  • Hooks can't be called inside loops, conditions, and nested functions,must be called in the same order and from react components.

Functional and Class components

  • Components are like JS functions, they accept inputs (called props) and return elements that described what should appear on the screen. .

  • Functional components are simple JS functions that accept props as an argument and return a React element.

  • A class component requires you to extend from React. Component and create a render function that returns a React element.

State

  • Is an object that determines how that component renders and behavior, in other words state is whats allows you to create components that are dynamic and interactive..

  • useState does exactly what it says—it allows us to use stateful values in function components..

  • Like all hooks, we call useState at the top of our component and can pass it an initial value to put on its state variable..

JSX

  • It is called JSX, and it is a syntax extension to JavaScript.

  • Is a syntax used as a JS extension in components to described how it is going to look like the UI.

  • JSX is used inside in our react component where you declare all the components that will be rendered, for this you can write any JavaScript expression between curly braces and be used.
